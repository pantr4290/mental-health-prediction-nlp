## How to Run
1. Install dependencies:
   pip install numpy pandas scikit-learn nltk
2. Open Jupyter Notebook:
   python -m notebook
3. Run the notebook in `notebooks/mentalhealth_model.ipynb`
# Mental Health Prediction using NLP

This project implements a supervised machine learning model to predict
depressive tendencies from Twitter text data.

## Approach
- Text preprocessing using NLP techniques
- Feature extraction with TF-IDF
- Classification using Logistic Regression

## Dataset
- Depressed tweets
- Non-depressed tweets
(Data collected from Twitter)

## Results
- Accuracy: ~86%
- Strong performance on both classes

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
- Jupyter Notebook
